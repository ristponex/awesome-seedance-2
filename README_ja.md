## Seedance 2.0 とは？

**Seedance 2.0** は ByteDance の次世代 AI 動画生成モデルで、**2026年2月8日〜12日**に中国国内の即梦（Jimeng）AIプラットフォームでリリースされました。AI動画生成におけるパラダイムシフトを象徴し、単純なテキストから動画への変換から、**シネマグレードの制作**へと進化しました。**同一の生成パスで同期された音声を生成**します。

前世代や競合モデルの多くがビデオとオーディオを別々の生成タスクとして扱うのに対し、Seedance 2.0は**音声・映像同時生成パイプライン**を導入。視覚コンテンツと同時に、自然に同期した効果音、環境音、対話、音楽を生成します。このシングルパスアプローチにより、AI生成マルチメディアに付きまとう不自然な時間的ずれを解消します。

### ビジョン

ByteDance は明確な目標を持って Seedance 2.0 を開発しました：**AI生成動画をプロの制作コンテンツと区別がつかないレベルにする**。以下の機能を備えています：

- **1回の生成で最大20秒**のクリップ
- **480p から 2K の解像度範囲**
- **ネイティブマルチショットストーリーテリング** — クリップを手動で繋ぎ合わせる必要なし
- **8言語以上のリップシンク**
- **物理的に正確なシミュレーション**
- **最大12個のリファレンスファイル**と直感的な @ タグシステム

Seedance 2.0 は段階的なアップデートではなく、世代を超えた飛躍です。

## 主要機能の詳細

### 1. 音声・映像同時生成

Seedance 2.0 のフラグシップ機能であり、市場の他のモデルとの最大の差別化ポイントです。

**仕組み：**
- 映像と音声がモデルの**単一のフォワードパス**で同時に生成
- 音声は映像生成後に「追加」されるのではなく、同時に作成される
- 視覚イベントとそのサウンドの完璧な時間的同期を保証

**生成される音声の種類：**
- **効果音** — ドアが閉まるフレームで正確にドアの閉まる音が鳴る
- **環境音** — 森のシーンには鳥のさえずりと風が木の葉を揺らす音
- **対話音声** — 自然な音声合成によるキャラクターの台詞
- **BGM** — 映像のムードとペースに合った背景音楽

**これが重要な理由：**
従来のアプローチでは：
1. 動画を生成
2. 動画内容を分析
3. 音声を別途生成
4. 音声を動画に合わせる（しばしば不完全）

Seedance 2.0 はステップ2〜4を排除し、最初から自然な映画的コンテンツを生成します。

```
プロンプト例：
「忙しいカフェでバリスタがミルクをスチームしている。エスプレッソマシンが
シューと鳴り、カップがソーサーの上でカチンと音を立て、柔らかなジャズが
バックグラウンドで流れている。カメラはバリスタの手から顧客の笑顔にゆっくり
パンする。」

結果：動画 + 完璧に同期されたカフェの環境音、マシンの音、BGM — すべて1回の生成で。
```

### 2. マルチショットネイティブストーリーテリング

従来のAI動画モデルは単一の連続ショットしか生成できませんでした。ストーリーを作るには、複数のクリップを個別に生成して手動で編集する必要があり、キャラクター、照明、スタイルの不一致が頻発しました。

**Seedance 2.0 はこれを根本的に変えます：**

- **単一のプロンプトで複数のショットを定義** — モデルは「カットする」「ディゾルブする」「次のシーン」などの映画用語を理解
- **自動的な一貫性** — ショット間でキャラクターの外見を維持
- **照明の連続性** — シーン間で照明が自然に変化
- **ペーシング制御** — モデルがナラティブのリズムを理解

**マルチショットプロンプト例：**
```
ショット1：夜のネオンに照らされた東京の通り、雨が降っている
ワイドエスタブリッシングショット。3秒。

ショット2：若い女性が傘を開き、ネオンサインを見上げる
ミディアムショット。水たまりに彼女の反射が見える。4秒。

ショット3：彼女が微笑む顔のクローズアップ。頬に雨粒、
瞳にネオンカラーの反射。3秒。

ショット4：背後からのワイドショット。傘を持って通りを歩き、
群衆の中に消えていく。5秒。
```

これにより、一貫したキャラクターの外見、自然なトランジション、全編を通して同期された雨音を持つ**15秒のミニフィルム**が生成されます。

### 3. @ リファレンスタグシステム

Seedance 2.0 で最も実用的なイノベーションの1つが、リファレンスファイル用の **@ タグシステム**です。

**仕組み：**
- 最大**12個のリファレンスファイル**をアップロード（画像、動画クリップ、スタイルリファレンス）
- プロンプト内で各リファレンスに **@ 識別子**をタグ付け
- モデルが指定した場所で各リファレンスを正確に適用

**サポートされるリファレンスタイプ：**
| リファレンスタイプ | 用途 | 例 |
|------------------|------|-----|
| キャラクターの顔 | シーン間のキャラクター一貫性 | @character_sakura |
| スタイルリファレンス | ビジュアルスタイルマッチング | @style_noir |
| 環境 | ロケーションの一貫性 | @location_office |
| オブジェクト | 特定のオブジェクト | @object_vintage_car |
| モーション | 動きのスタイルリファレンス | @motion_dance |
| 衣装 | 衣装の一貫性 | @outfit_kimono |

### 5. 物理シミュレーションエンジン

Seedance 2.0 には、AI生成動画にリアルな物理的インタラクションをもたらす物理シミュレーションエンジンが内蔵されています：

**サポートされる物理シミュレーション：**
- **流体力学** — 水しぶき、注ぐ液体、雨、波
- **布シミュレーション** — 布のドレープ、風による衣服への影響、カーテン
- **剛体物理** — 物体の落下、衝突、バウンド
- **軟体物理** — 変形可能なオブジェクト、ゼリー、ゴム
- **パーティクルシステム** — 煙、炎、火花、ほこり
- **髪の毛と毛皮** — 自然な動きと風の影響

### 6. リアルな人体モーション

Seedance 2.0 で最も目立つ改善は、人体の動きの処理です：

- **解剖学的に正確な**関節の動き
- **自然な歩行** — 歩く、走る、曲がるが人間らしい
- **手のディテール** — 指が自然に動く（AI動画の弱点）
- **顔の微表情** — 微妙な感情変化が可視
- **全身の協調** — 歩行中の自然な腕の振り、ターン時の重心移動
- **グループダイナミクス** — 同じシーンで複数の人物が自然にインタラクト

import requests

### テキストから動画生成（Seedance v1.5 Pro）

```python
import requests
import time

def generate_video(prompt, aspect_ratio="16:9"):
    """
    Atlas Cloud の Seedance v1.5 Pro で動画を生成。
    Seedance 2.0 では audio_enabled、multi_shot、
    reference_files などのパラメータが追加される予定。
    """
    url = f"{BASE_URL}/video/generations"

    payload = {
        "model": "seedance-v1.5-pro",
        "prompt": prompt,
        "aspect_ratio": aspect_ratio,
    }

    response = requests.post(url, json=payload, headers=headers)
    result = response.json()

    prediction_id = result["id"]
    print(f"生成開始：{prediction_id}")

    # 完了までポーリング
    while True:
        status_response = requests.get(
            f"{BASE_URL}/predictions/{prediction_id}",
            headers=headers
        )
        status = status_response.json()

        if status["status"] == "succeeded":
            print(f"動画完成：{status['output']['video_url']}")
            return status["output"]["video_url"]
        elif status["status"] == "failed":
            print(f"生成失敗：{status.get('error', '不明なエラー')}")
            return None

        time.sleep(5)

# 使用例
video_url = generate_video(
    "ゴールデンレトリバーが日差しの差す草原を走る、"
    "野の花が風に揺れる、シネマティックな照明、"
    "スローモーション、4K品質"
)
```

## プロンプトエンジニアリングのコツ

### 1. プロンプトを構造化する

```
[被写体] + [アクション] + [設定] + [照明] + [カメラ] + [スタイル] + [ムード]
```

**例：**
```
若い女性（被写体）が廃墟のボールルームで優雅に踊る（アクション）、
割れた窓から金色の日差しが差し込む（照明）、
ゆっくりとしたドリーイン（カメラ）、シネマティックなフィルムグレイン
（スタイル）、メランコリックで美しい（ムード）
```

### 2. 映画用語を使う

| 用語 | 効果 |
|------|------|
| 「ドリーイン/アウト」 | カメラが被写体に近づく/離れる |
| 「トラッキングショット」 | 動く被写体を追う |
| 「クレーンショット」 | カメラが上昇/下降 |
| 「ダッチアングル」 | 傾いたカメラで緊張感 |
| 「ラックフォーカス」 | 被写体間でフォーカス移動 |
| 「スローモーション」 | 再生速度を落とす |
| 「タイムラプス」 | 時間を加速 |
| 「アナモルフィック」 | ワイドシネマティック |
| 「浅い被写界深度」 | 背景ぼかし |

### 3. 照明を具体的に指定

```
良い：「ゴールデンアワーの逆光、レンズフレア、暖色系」
悪い：「きれいな光」

良い：「真上からの蛍光灯、グリーンティント、クリニカルな雰囲気」
悪い：「明るい光」
```

### 4. マルチショットプロンプト構造

```
ショット1：[時間] [ショットタイプ] [説明]
ショット2：[時間] [ショットタイプ] [説明]
ショット3：[時間] [ショットタイプ] [説明]
```

### 5. 音声プロンプティング（Seedance 2.0）

```
良い：「ブリキの屋根に雨が打ちつけ、遠くで雷が鳴り、猫が窓辺で
静かにゴロゴロ鳴いている。室内は暖かく居心地が良い。」
```

## よくある質問

**Q1：Seedance 2.0 とは何ですか？**

ByteDance の最新AI動画生成モデルで、2026年2月にリリースされました。シネマグレードの動画と同期音声を単一の生成パスで生成する初の主要モデルです。

**Q2：グローバルで利用できますか？**

まだです。2026年3月現在、中国の即梦AIプラットフォームでのみ利用可能です。[Atlas Cloud](https://www.atlascloud.ai?ref=JPM683) で **Seedance v1.5 Pro** は今すぐ利用できます。

**Q3：Seedance 2.0 はいつ Atlas Cloud で利用可能になりますか？**

ByteDance がグローバルAPIをリリース次第、Atlas Cloud が統合します。今すぐ登録して通知と優先アクセスを獲得しましょう。

**Q4：最大動画時間は？**

1回の生成で最大 **20秒**。マルチショットストーリーテリングにより、その時間内に複数のショットを含むナラティブを作成できます。

**Q5：リップシンクは何語をサポート？**

8言語以上：中国語、英語、日本語、韓国語、スペイン語、フランス語、ドイツ語、ポルトガル語。

**Q6：@ リファレンスタグの使い方は？**

リファレンスファイル（画像やビデオクリップ）をアップロードし、各ファイルに@タグを割り当てます。プロンプト内でそのタグを参照すると、モデルが対応するリファレンスを適用します。1回の生成につき最大12ファイル。

**Q7：Atlas Cloud は安全ですか？**

はい。SOC I & II 認証、HIPAA準拠、米国拠点の企業です。

**Q8：v1.5 のコードは v2.0 で動きますか？**

APIフォーマットは非常に類似する予定です。既存のv1.5コードは、音声生成やマルチショットなどの新機能用のパラメータ追加程度の変更で動作するはずです。

**Q9：料金はいくらですか？**

Seedance v1.5 Pro は $0.044/秒からで Atlas Cloud で今すぐ利用可能。2.0 は 720p で約$0.10/分〜2K で約$0.80/分と推定されています。

*表示価格は最低価格です。より高い解像度や長い動画は追加料金が発生する場合があります。*

**Q10：紹介ボーナスの使い方は？**

[紹介リンク](https://www.atlascloud.ai?ref=JPM683)から登録すると、初回入金時に **25%ボーナス（最大$100）**を受け取れます。
